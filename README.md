# WhatsApp AI Agent

Agente de IA para qualificação de leads e atendimento humanizado no WhatsApp.

## Stack
- FastAPI + Python
- Claude Haiku (conversas) + Claude Sonnet (análise)
- Redis + Celery (fila de mensagens)
- Supabase (banco de dados)
- Evolution API (WhatsApp)

## Deploy
Railway — veja `railway.json` e `Procfile`
